Global Network Map

Batadv-vis is used to visualize our batman-adv mesh network. It reads the neighbor information and local client table and distributes this informatoin via alfred in the network. By gathering this local information, any vis node can get the whole picture of the network and the link quality between devices.

Note: the link quality is inversely proportional to the number displayed.
